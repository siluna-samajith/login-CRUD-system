# PHP Authentication & Admin Dashboard System

A simple and secure web application built with PHP and MySQL that includes user authentication, role-based access control, and an admin dashboard.

##  Features

- User Login & Logout system
- Secure password hashing (bcrypt)
- Role-based access control (Admin / User)
- Admin Dashboard
- User Management (CRUD operations)
- Protected routes
- Clean UI using Tailwind CSS

##  Technologies Used

- PHP (Core PHP)
- MySQL
- Tailwind CSS

##  Authentication

- Passwords are hashed using `password_hash()`
- Login verified using `password_verify()`
- Sessions used for authentication

##  Admin Features

- View total users
- Add new users
- Edit users
- Delete users

##  User Features

- Login system
- Redirected to user home page
- Restricted access to admin panel
